# koishi-plugin-ksak-tmp

[![npm](https://img.shields.io/npm/v/koishi-plugin-ksak-tmp?style=flat-square)](https://www.npmjs.com/package/koishi-plugin-ksak-tmp)

阿K传媒机器人TruckersMP查询插件 - 支持玩家查询、服务器状态、路况信息、Steam打折通知等功能

## 🚛 插件介绍

欧洲卡车模拟2线上插件TMP查询机器人

## 📋 指令说明

| 指令名称 | 指令介绍 | 使用示例 |
|---------|---------|---------|
| 绑定 | 绑定 TMPID，绑定后使用其他指令时可省略输入 | 绑定 123 |
| 查询 | 查询TMP玩家信息，支持@查询他人 | 查询 123 或 查询 @某人 |
| 定位 | 查询玩家位置信息，支持@定位他人 | 定位 123 或 定位 @某人 |
| 路况 | 查询服务器热门地点路况信息，支持服务器简称：s1、s2、p、a | 路况 s1 |
| 欧卡服务器 | 查询欧卡服务器信息列表 | 欧卡服务器 |
| 美卡服务器 | 查询美卡服务器信息列表 | 美卡服务器 |
| 插件版本 | 查询插件版本信息 | 插件版本 |
| 插件规则 | 返回TruckersMP官方规则链接 | 插件规则 |
| 地图dlc | 查询地图DLC列表 | 地图dlc |
| 总里程排行 | 总里程排行榜，绑定ID后可查看自己的排名 | 总里程排行 |
| 今日里程排行 | 今日里程排行榜，每日0点重置数据 | 今日里程排行 |
| 足迹 | 今日足迹，查询除P服外的所有服务器足迹 | 足迹 123 |
| 足迹p | 今日足迹，查询P和PA服的足迹 | 足迹p 123 |
| 查询vtc | 查询虚拟卡车公司信息 | 查询vtc 123 |
| 查询活动 | 查询TruckersMP活动信息 | 查询活动 |
| 查询游戏价格 | 查询欧卡和美卡Steam价格 | 查询游戏价格 |
| 欧卡打折 | 查询欧卡2 Steam打折活动 | 欧卡打折 |
| 美卡打折 | 查询美卡 Steam打折活动 | 美卡打折 |
| steam打折管理 | 管理Steam打折通知订阅 | steam打折管理 |
| 欧卡打折设置 | 开关欧卡打折通知 | 欧卡打折设置 |
| 美卡打折设置 | 开关美卡打折通知 | 美卡打折设置 |
| 解绑 | 解绑已绑定的TMP ID | 解绑 |

## ⚙️ 配置选项

### 基本配置
- 百度翻译功能（需要配置百度翻译API）
- Steam API Key（用于查询游戏运行时间）

### 指令配置
- 用户头像显示开关（谨慎使用，部分玩家头像可能违规）
- VTC信息显示开关
- 欧卡/美卡运行时间显示开关
- 路况信息展示方式（文字/热力图）

### 版本更新通知
- 启用TMP版本更新通知
- 检查间隔（分钟）
- 接收通知的群号列表

### 活动提醒
- 启用TMP活动提醒
- 检查间隔（分钟）
- 接收活动提醒的群号列表

### Steam打折通知
- Steam价格检查间隔（分钟）
- 接收打折通知的群号列表
